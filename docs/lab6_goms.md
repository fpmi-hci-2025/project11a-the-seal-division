# Анализ интерфейса по модели GOMS
### Базовые операторы GOMS (время в секундах):
- K (Keystroke): Нажатие клавиши = 0.28 сек
- P (Point): Указание мышью = 1.1 сек
- B (Click): Щелчок мышью = 0.1 сек
- H (Home): Перемещение рук = 0.4 сек
- M (Mental): Ментальная подготовка = 1.35 сек  
### Форма входа состоит из 2х полей: имя пользователя, пароль + кнопка
### Форма регистрации состоит из 5 полей: имя пользователя, номер телефона, почта, пароль и повтор пароля + кнопка
### Форма оплаты корзины состоит из 4 полей: имя, номер карты, дата, код + кнопка  
**Примем в среднем что в каждом поле пользователь вводит 10 символов**
